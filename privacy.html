Privacy Policy for Just Focus

Effective Date: Dec 22, 2024

Data Collection

Just Focus does not collect any personal data or send user data to external servers. The extension stores the following data locally in the browser:

  • Task: The task description set by the user.
  • Progress: The progress made towards completing the task.
  • Total: The target goal set for the task.

This data is stored using the browser's localStorage, and it remains only on the user's device. The data is not shared, sold, or transferred to third parties.

Data Usage

The data stored in localStorage is used solely for the purpose of tracking task progress and ensuring that the information is preserved even after the user refreshes or reopens the extension.

Data Retention

Data is retained locally and can be modified or deleted by the user at any time. The extension does not access or store this data in any central database.

No Third-Party Access

No third-party services have access to the data stored in localStorage. This extension does not collect any personally identifiable information.
